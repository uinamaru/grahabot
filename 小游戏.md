### 限额说明
>[warning]通过互动可以获得的好感值有限额，每获得【15次】需要冷却cd【1小时】，而通过小游戏获得的好感同样会计算在限额次数里（但商店道具加的好感不算）。<strong>超过次数依然可以玩但是无法获得好感。</strong>
通过小游戏获得的金币也有限额，每日合计可以玩【5次】小游戏，同一种游戏玩5次或者不同游戏合计玩5次均可。<strong>超过次数无法开启小游戏，猜拳不限制次数。</strong>

```
```

```
```

